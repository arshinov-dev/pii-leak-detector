# Tabular PII Detector

Детектор персональных данных (ПДн) в табличных файлах для интеграции в классификатор рисков утечек. Проект создан для хакатона и ориентирован на обнаружение российских типов ПДн.

## Возможности

- **6 типов обнаруживаемых ПДн:**
  - ИНН (10 и 12 знаков) — с математической проверкой контрольной суммы
  - СНИЛС (11 цифр) — с математической проверкой контрольной суммы
  - Паспортные данные (серия + номер)
  - Номера телефонов (российские форматы: +7 и 8)
  - ФИО — через NLP-модель Natasha
  - Адреса — через NLP-модель Natasha

- **Контекстный анализ** для снижения ложных срабатываний (проверка названий колонок и соседних ячеек)
- **Безопасное маскирование** найденных ПДн в отчёте (в логи не попадают реальные данные)
- **Многостраничные Excel-файлы** — обрабатываются все листы
- **Автоопределение кодировки CSV** (UTF-8 → CP1251)
